# Project Blueprint: Dream Interpretation AI Lotto

## Overview
이 프로젝트는 사용자의 꿈과 직관을 바탕으로 로또 번호를 추천해주는 감성적인 웹 앱입니다. GitHub에 소스를 저장하고 Cloudflare Pages를 통해 배포합니다.

## Features & Design
- **Core:** Framework-less HTML5, CSS3 (Modern Gradient & Glassmorphism), ES Modules.
- **Theme:** "Dream Interpretation" (꿈해몽), Mystical, Emotional.
- **Deployment:** Cloudflare Pages.
- **Source Control:** GitHub.

## Current Plan & Steps
1. [ ] **GitHub 연결:** 로컬 git 저장소 설정 및 커밋.
2. [ ] **Cloudflare Pages 설정:** GitHub 저장소를 Cloudflare Pages와 연동.
3. [ ] **배포:** 코드를 GitHub에 푸시하여 자동 배포 트리거.

## Implementation Details
- Entry Point: `index.html` (Single Page App)
