# 🌙 꿈해몽 AI 로또 추천 (Dream AI Lotto)

> **"당신의 무의식이 보내는 행운의 신호, AI가 해석해 드립니다."**

## 소개
어젯밤 당신의 꿈에는 엄청난 행운이 숨어있을지 모릅니다. **꿈해몽 AI 로또 추천**은 사용자의 꿈 내용을 AI가 분석하여 그 속에 숨겨진 행운의 숫자를 찾아주는 감성적인 웹 애플리케이션입니다.

## 주요 기능
- **✨ 꿈 일기 (Dream Input):** 사용자가 자신의 꿈을 자유롭게 서술할 수 있는 입력 창.
- **🔮 AI 꿈 분석 시뮬레이션:** 입력된 텍스트에서 키워드를 추출하여 행운 메시지를 제공합니다.
- **🔢 커스텀 번호 조합:** 꿈에서 본 숫자 포함하기, 피하고 싶은 숫자 제외하기 등 개인화된 추출이 가능합니다.
- **📱 반응형 디자인:** 모바일과 웹 어디서나 아름다운 UI를 제공합니다.

## 기술 스택
- **Frontend:** HTML5, CSS3 (Modern Baseline), JavaScript (ES Modules)
- **Styling:** Glassmorphism, `oklch` color spaces, `@container` queries
- **Deployment:** Cloudflare Pages

## 설치 및 실행
1.  이 저장소를 클론합니다.
2.  `public/` 디렉토리 내의 `index.html`을 브라우저에서 엽니다.
3.  또는 Cloudflare Pages를 통해 배포합니다: `npx wrangler pages deploy public`
